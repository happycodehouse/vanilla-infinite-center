## Features
- **Infinite Loop**: Clone-based infinite scrolling with seamless transitions
- **Center Focus**: Center slide is wider than surrounding slides
- **Mask Animation**: Clip-path reveal effect synced with slide transitions
- **Responsive Design**: Adapts layout based on screen width (mobile / desktop)
- **Pagination**: Jump to any slide with step-by-step animation

## Tech Stack
- **GSAP** (animations)
- **JavaScript (ES6+)**
- **SCSS**

## Variations
| Type | Description |
|------|-------------|
